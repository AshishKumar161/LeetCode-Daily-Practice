# LeetCode 206 - Reverse Linked List

## Question

Given the `head` of a singly linked list, reverse the list and return the reversed list.

## Example 1

Input:

```text
head = [1,2,3,4,5]
```

Output:

```text
[5,4,3,2,1]
```

## Example 2

Input:

```text
head = [1,2]
```

Output:

```text
[2,1]
```

## Example 3

Input:

```text
head = []
```

Output:

```text
[]
```
