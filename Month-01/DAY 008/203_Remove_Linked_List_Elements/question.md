
# LeetCode 203 - Remove Linked List Elements

## Question

Given the `head` of a linked list and an integer `val`, remove all the nodes of the linked list that have `Node.val == val`.

Return the new head of the linked list.

---

## Example 1

Input:

```text
head = [1,2,6,3,4,5,6]
val = 6
```

Output:

```text
[1,2,3,4,5]
```

---

## Example 2

Input:

```text
head = []
val = 1
```

Output:

```text
[]
```

---

## Example 3

Input:

```text
head = [7,7,7,7]
val = 7
```

Output:

```text
[]
```
