# LeetCode 75 - Sort Colors

## Question

Given an array `nums` containing only `0`, `1`, and `2`, sort the array in-place.

The values represent colors:

- `0` = Red
- `1` = White
- `2` = Blue

You must not use the built-in sort function.

## Example 1

Input:

```text
nums = [2,0,2,1,1,0]
```

Output:

```text
[0,0,1,1,2,2]
```

## Example 2

Input:

```text
nums = [2,0,1]
```

Output:

```text
[0,1,2]
```
