
# LeetCode 19 - Remove Nth Node From End of List

## Question

Given the `head` of a linked list, remove the `n-th` node from the end of the list and return its head.

---

## Example 1

Input:

```text
head = [1,2,3,4,5]
n = 2
```

Output:

```text
[1,2,3,5]
```

---

## Example 2

Input:

```text
head = [1]
n = 1
```

Output:

```text
[]
```

---

## Example 3

Input:

```text
head = [1,2]
n = 1
```

Output:

```text
[1]
```

---
